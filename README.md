# Simple Todo en Java

**Auteur :** Nicolas CHHIM
**Projet personnel**

## Description
Projet personel réalisé lors de mon année de L1 Informatique pour apprendre les bases de Java.
Le programme fonctionne en console et permet de gérer des tâches simplement en les enregistrant dans un fichier, pour pouvoir les retrouver à chaque lancement.

## Fonctionnalités
- Ajouter une tâche
- Afficher les tâches
- Menu interactif

## Langage de programmation
- Java

## Exécution
javac SimpleTodo.java
java SimpleTodo
